Insertar imágenes

Con IMG las inserto. Tiene varios atributos:

ALT = "Pongo texto alternativo por si no se carga la imágen"

SRC = "ruta (URL) de la imágen"

WIDTH = "150px" así tal cual se pone

Si quiero cargar imágenes directo de la pc en src pongo la ruta de la imágen. Si está en la misma carpeta que el index solo pongo el nombre
